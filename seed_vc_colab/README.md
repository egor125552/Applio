# Seed-VC V2 Colab

Минимальный Google Colab для zero-shot voice conversion обычной речи через готовый Gradio-интерфейс.

## Запуск

1. Открой блокнот `Seed_VC_V2_Colab.ipynb` в Google Colab.
2. Выбери GPU: **Среда выполнения → Сменить среду выполнения → T4 GPU**.
3. Нажми **Выполнить всё**.
4. После загрузки моделей в последней ячейке появится публичная ссылка Gradio.
5. В интерфейсе загрузи `Source Audio` и `Reference Audio`, затем запусти конвертацию.

Обучение не требуется. Референс лучше брать чистый, без музыки и реверберации, длиной 10–25 секунд.

## Проверки

Тесты не добавлены в ячейки блокнота. Они выполняются отдельно в GitHub Actions:

- проверка структуры и чистоты notebook;
- проверка установочных и запускающих скриптов;
- установка CPU-окружения с теми же Python-зависимостями;
- импорт Seed-VC V2 и его модельных модулей;
- проверка Hydra-конфигурации;
- smoke-тест запуска локального Gradio-сервера;
- проверка существования удалённых checkpoint-репозиториев и файлов без скачивания гигабайтов весов.
